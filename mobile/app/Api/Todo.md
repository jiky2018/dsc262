=======

1. 首页(店铺)

2. 分类

3. 购物车

4. 我的

5. 商品列表

6. 商品详情

7. 订单提交

会员中心
1. 订单列表（待支付、待收货、待评价、全部订单）（订单详情）

2. 我的钱包（）

3. 收货地址

4. 我的收藏

5. 我的优惠券


参考项目：

1. 酷客多

2. 平台接口需要升级为https
3. 进入后要求先关联手机号码
